按照CSS样式书写的位置可分为三大类：1.行内样式表（行内式），2.内部样式表（嵌入式），3.外部样式表（链接式）

行内样式表：

在元素标签内部的style属性中设置CSS样式，使用于简单的样式修改

 < div style="color：red; font-size: 12px;"> 青春不常在，抓紧谈恋爱 < /div >

内部样式表：

写在html页面内部，将所有的CSS样式抽离出来放到< style >标签中

   <head>
            <style>
                p {
                    text-indent: 2em;
                }
                pre {
                    text-indent: 2em;
                    background-color: aqua;
                }
            </style>
      </head>

外部樣式表：

样式单独写在一个CSS文件中，之后把CSS文件引入到html页面中使用

 
        1.新建一个后缀名为.css的样式文件，把所有css样式代码都放到这个文件中
        2.在html页面中使用<link>标签引入这个文件
    
        < link rel="stylesheet" href="css文件路径">
