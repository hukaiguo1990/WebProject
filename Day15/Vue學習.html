创建Vue例子

1.引入vue版本
开发环境： 生产环境b: 2.创建简洁的模板
{{ message }}
3.创建vue实例 var app = new Vue({ el: '#app', data: { message: 'Hello Vue!' } })

el：挂载点

  1. el是用来设置Vue实例挂载的元素
  2. Vue会管理el属性命中的元素及其内部的后代元素
  3. 可以使用类选择器，标签选择器，建议是用ID选择器
  4. 模板可以使用其他的标签，但不能使用HTML和body，建议使用div

data:数据对象

  1. Vue中用到的数据定义在data中
  2. data中可以写复杂类型的数据

Vue指令

  1. v-text: 设置标签的内容
  2. v-html: 设置元素的innerHTML,内容中有html结构会被解析成标签
  3. v-on: 为元素绑定事件，可以用@代替
  4. v-show:根据条件的真假切换元素的显示状态，原来是修改元素的display实现元素的显隐
  5. v-if:根据条件的真假切换元素的显示状态，和v-show不同的是v-if是操作dom元素切换状态的，性能方面:切换频繁的话建议使用v-show，反之v-if
  6. v-bind: 简写为":" 为元素绑定属性值，如果是class属性建议使用对象的方式
  7. v-for: 根据数据生成列表结构；语法是v-for: (item, index) in arr
  8. v-model: 便捷的设置和获取表单元素的值，绑定的数据会和表单元素之相关联：双向绑定，不管修改那一方，另一方也会同步变更
