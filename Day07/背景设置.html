background-color:设置背景颜色

用法： 
        div {
            background-color: #red;
        }

background-image:设置背景图片

用法：必须要用url方式 
        div {
            background-image: url(image/a.png);
        }

background-repeat:设置背景平铺

用法：参数有repeat背景平铺,no-repeat背景不平铺,repeat-x：背景沿x轴平铺,repeat-y背景沿y轴平铺 
        div {
            background-repeat: repeat;
        }

background-position:设置背景图片的位置

用法：position后可以跟方位名词（top,bottom,left,right,center）,也可以跟精确的数值;如果是精确数值时，一个参数必须是x轴的，第二参数是y轴的，顺序不能替换 
        div {
            background-position: center;
        }

background-attachment:设置背景滚动

用法：scroll(滚动)，fixed(固定的) 
        div {
            background-attachment: fixed;
        }

背景色半透明：background: rgba(0,0,0,0.5)

用法: 
        div {
            background: rgba(0,0,0,0.5);
        }
