定位组成

定位：将盒子定在某个位置；所以定位也是摆盒子，按照定位的方式移动盒子
定位=定位模式+边偏移
我是粘性定位sticky

定位模式：

通过position属性来设置，值有：
  1. static: 静态定位
  2. relative: 相对定位
  3. absolute: 绝对定位
  4. fixed: 固定定位

边偏移

就是定位的盒子移动到的最终位置，有top,bottom,left,right属性。

静态定位：static

静态定位按照标准流特性摆放位置，它没有边偏移；很少用到

相对定位：relative

相对自身的原有位置进行偏移；
偏移后，原有位置还会保留占有，后面的盒子依然以标准流的方式对待它（不脱标，继续保留原来位置）

绝对定位: absolute

  1. 如果祖先元素或者祖先元素没有定位，则以浏览器为基准进行定位
  2. 如果祖中有定位，则以最近一级的有定位的祖先元素为基准进行定位
  3. 绝对定位不再占有原先的位置（脱标）

固定定位 fixed

  1. 以浏览器的可视窗口为参照点进行移动元素的
  2. 跟父元素还没有任何关系
  3. 不随滚动条滚动
  4. 脱标，固定定位不再占有原来的位置

粘性定位：sticky

  1. 以浏览器的可视窗口为参照点进行移动元素（固定定位特点）
  2. 粘性定位占有原先的位置
  3. 必须设置top,left,right,bottom中的一个才会生效
  4. 兼容性差，IE不支持
